Tutorial


Starting a new study sessionssss

To start a new study session, simply set a time and a focus and click Start. The timer is displayed in hours:minutes:seconds so if you want to study for an hour and 5 minutes, simply type "10500". Next, set a focus and click the start button to begin your study session. During this period, we will block any sites unrelated to your selected focus.

Whitelisting

Whitelisting allows you to set a list of sites that you don't want to be blocked. You should whitelist school sites like Canvas or Schoolloop. To add a whitelist site, click the page icon at the bottom and select Whitelist. Next, type in the full url (ex. mail.google.com, drive.google.com, www.google.com) into the text input and click Add.

Subjects

In the subject dropdown, you can select several focuses. Currently, there are only a few accessible subjects like Biology and College apps but more will be added in the future. You can also select the General Studying option, which will only allow you to access whitelisted sites.

Unblock

In the event that we mistakenly block a site that shouldn't be blocked, simply click the Unblock button at the bottom to gain access quickly (note: unblocked sites will not be whitelisted and will be only be temporarily accessible).

Settings

To adjust settings, click on the gear icon and select Settings. This page will allow you to adjust how strict the blocking is and turn the timer widget on or off. Adjust the settings and click the Save button to save your settings.
